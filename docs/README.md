# Benefit Guard
### Описание
**Benefit Guard** - это чат-бот в отечественном мессенджере *VK MAX*, миссия которого заключается в помощи людям, имеющим инвалидность, по поиску доступных им субсидий.
### Сборка и запуск
Чат-бот реализован с помощью программного обеспечения *docker*, что позволяет выполнить развёртывание бота на любой системе! Достаточно лишь иметь установленный *docker*.
